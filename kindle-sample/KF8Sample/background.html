Background Styles

This chapter describes two types of background styles:

  • Background color
  • Background image

Background styles helps to make the content more appealing.

Background Color

The content can have any color as a background color. The example below has multiple background colors.

Example:

This block has a dark sea green color background

This block has a yellow chiffon background

This block has a gray background

End of yellow chiffon background

End of dark sea green background

How To:

To set the background of the content, use the background CSS property.
background:red;
