Text Styling

KF8 supports a variety of ways to style the text of a book.

Font Styles

In general, do not specify the font style. Users prefer to select their own default reading settings.

Font tag

Font face can be set. Users will see this font instead of their selected font. In addition to the available fonts, custom fonts can be added and used.

Example:

Georgia, Caecilia, Trebuchet, Verdana, Arial, Times New Roman, Courier, Lucida
Red size 1 text. Blue size 2 text. Green size 4 text. Purple size 6 text.

How To:

To set the font, use the font tag and its face attribute. To set the color and size, use the color and size attributes.
<font face="Georgia" color="green" size=4 >
Font face can also be set by using the font-family CSS styling property.
font-family: arial;
color: red;
font-size: 120%;

CSS font styles

Font styles such as style, weight, and variant can be set by using CSS styling properties.

Example:

Normal text.
Italic lighter text.
Italic bold text.
Oblique bold small-caps text.

How To:

To set these font styles, use the font-style, font-weight, and font-variant properties.
<span style="font-style:italic; font-weight: lighter; font-variant: normal">

Text Styling

The content can be formatted with other text styling, such as word spacing and text align.

Example:

Uppercase text with Word spacing 1em.
Underlined text with letter spacing 0.2em, word spacing 2em.
Capitalized line through text.

How To:

To set this text styling, use the word-spacing, text-transform, letter-spacing, and text-decoration CSS properties.
<span style="letter-spacing: .2em; word-spacing: 2em; text-decoration:underline;">

Right aligned text.

How To:

To set text alignment, use the text-align property. To set text indentation, use the text-indent property.
text-align:right;
text-indent:0em;

Shadowed text.

margin-bottom: -.2em;

padding-right: .1em;
}

The dimension of the content display can be set.

Example:

Lorem ipsum duis ultrices eleifend rhoncus. Quisque vulputate nisl ultricies nisi posuere at lacinia quam suscipit. Curabitur risus nunc, laoreet at interdum eget, rutrum non neque. Phasellus varius orci fermentum mi aliquam vitae convallis odio aliquet. Nam commodo lobortis sem, vel ultricies erat suscipit eget.

How To:

To set the content display dimensions, use the width and height CSS properties.
width:50%; height:50%;
While setting dimensions, some words could exceed the display width. The words can be wrapped by using the word-wrap property.
word-wrap:break-word;
