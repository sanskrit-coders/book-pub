Image Styling

Different styles can be applied to the images with CSS styling properties.

Image border

Images can have different styles of border. Border styling can vary with size, shape, etc.

How To:

To add border to an image, use the border attribute. The borders of the images above are set with values of 1, 2, and 3, respectively.

Border colors

Image borders can be set to different colors using CSS styling properties.

Example:

How To:

To set the color of the border, use the border-color CSS property.
style="border-color:red"

Rounded border

A rounded border for an image can be set by using CSS styling properties. By adjusting CSS property values, different shapes of borders can be achieved. Changing the values of other properties changes styles such as background color and border style.

Example:

How To:

To set these types of styles, use the border, padding, background, and border-radius CSS properties.
img.border1
{
border:.2em solid #a1a1a1;
padding:10px 10px;
background:#dddddd;
border-radius:10px;
}

Image Transformation

An original image can be transformed to a different style by using CSS styling properties. The size of the image can be changed to fit the content.

Image resizing

An image can be displayed in any size by specifying the display size.

How To:

Image rotation

An original image can be rotated to any angle by using CSS styling properties.

Example:

How To:

In this example, a single image file is displayed in different sizes by using the width and height attributes of the img tag. To rotate and place the image, two div tags are used with CSS styling. One div tag is used to place the image at the correct position by using the translate CSS function. Another div tag is used to rotate the image by using the rotate CSS function.
<div style="-webkit-transform : translate(0px,150px); ">
<div style="width:10px; height:10px; -webkit-transform : rotate(216deg); float:left; ">
<img src="images/tlook.jpg" width=10 height=10 border=1></img>
</div>
</div>
